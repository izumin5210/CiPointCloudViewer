## Getting started
### Setup

1. Clone this repository
2. Run `./scripts/setup.sh`

### Build this app

```
$ cd proj/cmake

$ mkdir build && cd build

$ cmake  ..

$ make -j$(nproc)

$ ./Debug/CiPointCloudViewer
```

or open by CLion.
